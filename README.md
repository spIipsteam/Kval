## Описание

«АЭРКОМ-Сервис» — это веб-приложение для автоматизации учёта услуг фирмы по обслуживанию домофонов, разработанное в рамках производственной практики в учебном заведении. Приложение позволяет сотрудникам компании систематизировать данные о клиентах и обслуживаемых объектах, отслеживать заявки на ремонт, контролировать работу мастеров и вести складской учёт в удобной цифровой форме.

## Установка

Убедитесь, что на компьютере установлен любой современный браузер (Chrome, Firefox, Edge последних версий)

Скачайте файл приложения: aercom-service.html

Откройте файл в браузере двойным щелчком мыши

При первом запуске автоматически создадутся тестовые данные в localStorage браузера

Для входа в систему используйте учётные данные:
- Менеджер: логин `manager`, пароль `manager123`
- Мастер: логин `master`, пароль `master123`

## Использование

«АЭРКОМ-Сервис» — это удобное и функциональное веб-приложение для автоматизации работы сервисной службы, построенное на современных веб-технологиях с хранением данных в браузере. Оно предоставляет полный инструментарий для управления клиентской базой, обработки заявок, контроля работы мастеров и учёта материалов на складе. Приложение имеет два режима доступа: менеджер управляет всеми процессами, мастер видит только свои задачи и может отмечать их выполнение.
